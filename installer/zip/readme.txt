==================================================================
README
==================================================================

Welcome to GeoNetwork.

To run GeoNetwork, you need the Java Runtime Environment (JRE)
version 1.5.0. You can download it from this url:

http://java.sun.com/javase/downloads/index_jdk5.jsp

==================================================================
WINDOWS USERS
==================================================================

Under Windows, you can use the provided Setup.exe file which will 
assist you even if you don't have Java VM installed.

==================================================================
OTHER USERS
==================================================================

Once you have downloaded the JRE, simply click on the .jar icon
to launch the installer. If this does not work, open a console
and issue:

java -jar geonetwork-xxx-install.jar

==================================================================
