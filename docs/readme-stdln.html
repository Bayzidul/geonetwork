GeoNetwork opensource V2.1.0 alpha 1 Date: 16-10-2006

GeoNetwork desktop release notes

Introduction

Welcome to GeoNetwork opensource Spatial Data Catalogue system. The desktop installation has been developed to allow a very easy installation, configuration and start up of the GeoNetwork opensource software for use on a Personal Computer. For server installations, please use the server specific installers.

The installer is configured to run using an embedded database, McKoi and on an embedded web server, Jetty.

Run the installer and choose to install the Java Runtime Environment (JRE 1.5) if none is present on your PC yet. The JRE is provided in the full version of the installer or can be downloaded separately (see Requirements below) . The installer will create a "GeoNetwork desktop" group of shortcuts in the Start menu under Programs. You can start the web server from there and also open the home page of your local GeoNetwork opensource installation from here.

!!! Very important !!!

Before you try to open the GeoNetwork desktop homepage in a browser, make sure you first start the web server! You do this by selecting the "Start server" from the start menu. Startup of the server requires a little bit of time (usually less than a minute). You can monitor the web server output by clicking on the "Start server" application icon that appeared in the toolbar.

To start editing metadata, log in with the username and password you provided during the installation.

To stop the server, use the "Stop server" shortcut in the start menu.

Advanced features of the system, like metadata harvesting from other nodes, are described in detail on the community web site (see below).

Requirements

Java

A Java Runtime Environment (JRE) 1.5 Standard Edition (download from http://java.sun.com) is required to run GeoNetwork opensource. See the Troubleshooting sections when normal installation or start up fails.

Web server

This installation is configured to run on the embedded Jetty web server and uses port 8080 by default. You can open the GeoNetwork opensource homepage in a web browser by using the following URL (on the same computer!): http://localhost:8080/geonetwork

You may wish to change this manually in the Jetty configuration file (by default in c:\Program files\geonetwork_desktop\bin\jetty.xml) and the GeoNetwork opensource configuration file (c:\program files\geonetwork_desktop\web\WEB-INF\config.xml). Also the file ..\bin\win\start-stdln.html will need to be updated. You can open all of these files in a text editor.

Database system

GeoNetwork desktop comes with an internal DBMS server, the McKoi SQL database. If you plan to use another DBMS (like Oracle or MySQL) please install the server version of GeoNetwork opensource and carefully read the documentation.

Other operating systems

Although primarily created for PCs running Windows, the installer will also work on other operating systems. A Java Runtime Environment (JRE) 1.5 has to be installed first. You can start the installer from the command line/ terminal window using the following command:

java -jar geonetwork-intermap-2.1.0_alpha_1-stdln.jar
