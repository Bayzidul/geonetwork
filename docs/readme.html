GeoNetwork - Getting started

Running the application

You have to launch the web server before starting the GeoNetwork opensource application: Navigate to the start-menu folder you created during installation, then launch "start server". You can then open the GeoNetwork application in your browser.

By default this should be at http://localhost:8080/geonetwork

A Java Runtime Environment (JRE) 1.5 Standard Edition (download from http://java.sun.com) is required to run GeoNetwork opensource. See the Troubleshooting sections if normal installation or start up fails.

IMPORTANT!! After opening the GeoNetwork homepage, log on as Administrator. You should use the username admin and password admin. In the Administration section you can then modify your account.

Loading sample data

The catalog comes with metadata templates setup. However, no sample metadata has been loaded by default.

If you want to load sample metadata, you have to start the GeoNetwork Administrator Survival Tool (GAST) tool.

  • Configure your authenthication in "Options" - "Config"
  • Select the Sample data option in the Database section and press Import

You are now ready to test with sample metadata records.

Features

Web server

The installer comes with an embedded Jetty web server but you can consider using Jakarta Tomcat instead (see post-install notes).

You can open the GeoNetwork opensource homepage in a web browser by using the following URL http://<your server URL>:<your port>/geonetwork. If you install on your local machine using port 8080 (default), you could use http://localhost:8080/geonetwork

Database system

GeoNetwork desktop comes with an internal DBMS server, the McKoi SQL database.

If you plan to use another DBMS please select that DBMS using the GAST tool and make sure that the DBMS is already up and running. Oracle, MySQL and PostgreSQL are currently supported

You can also configure the DBMS manually at a later stage, although this is more complex (See the community website for details)

More information at http://geonetwork-opensource.org
